## 🧠 谜题的存储方式

所有数独谜题以 **81 位数字字符串** 的形式硬编码在 `sudoku.html` 中。每个字符串表示一个按行优先排列的 9×9 网格，其中 `0` 代表空格。  
例如：  
`"208010570..."` 对应 →  
```
2 0 8 0 1 0 5 7 0
4 0 1 3 5 0 0 0 0
...
```

内置的回溯求解器保证每道谜题**有且仅有一个解**。
